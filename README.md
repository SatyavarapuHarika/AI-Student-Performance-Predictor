# 🎓 AI Student Performance Predictor

A full-stack Machine Learning web application that predicts student marks based on study hours and attendance.

---

## 🚀 Features

* 📊 Predict student performance using Machine Learning
* 🧠 Random Forest Regression model
* 🌐 Flask backend with REST API
* 🎨 Modern UI with animations & glassmorphism
* 🌙 Dark/Light mode toggle
* 📈 Data visualization using Chart.js

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Flask
* **Machine Learning:** Scikit-learn
* **Visualization:** Chart.js

---


## ⚙️ Run Locally

```bash
git clone https://github.com/SatyavarapuHarika/AI-Student-Performance-Predictor.git
cd AI-Student-Performance-Predictor

pip install -r requirements.txt
python model.py
python app.py
```

---

## 🌐 Usage

1. Open browser
2. Go to: http://127.0.0.1:5000/
3. Enter inputs
4. Get prediction instantly
